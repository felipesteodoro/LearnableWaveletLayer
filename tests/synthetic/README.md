# Track 1 — Synthetic Signal Regression

**Task**: reconstruct a clean signal from a noisy composite waveform.  
**Goal**: evaluate whether the LWT layer can adapt its filters to the multi-scale structure of a known signal better than fixed wavelet baselines (db2, db4).

---

## Dataset

Synthetic time series generated by `src/data_generator.py` with the following components:

| Component | Description |
|---|---|
| Trend | Polynomial (degree 2) |
| Harmonics | 5 sinusoidal components |
| Chirp | Frequency-swept sinusoid |
| Regime changes | 3 abrupt amplitude shifts |
| Transients | Localised burst events |
| Noise | Additive Gaussian (level = 0.3) |
| Spikes | Random impulses (probability = 0.02) |

- **Total samples**: 50,000
- **Sequence length**: 256
- **Split**: 50% train / 30% validation / 20% test

## Models

**ML** (wavelet features → scalar vector → regressor):
- LinearSVR, SGDRegressor, ElasticNet
- RandomForest, XGBoost, LightGBM, CatBoost, Stacking

**DL** (three input modes):

| Mode | Input |
|---|---|
| `raw` | Raw signal sequence |
| `db2` | Fixed db2 wavelet coefficients (via `FixedDb4DWT1D`) |
| `learned_wavelet` | Coefficients from `LearnedWaveletDWT1D_QMF` |

Architectures: CNN · LSTM · CNN-LSTM · Transformer

---

## Metrics

| Metric | Description |
|---|---|
| MSE / RMSE | Mean squared / root mean squared error |
| MAE | Mean absolute error |
| MAPE | Mean absolute percentage error |
| R² | Coefficient of determination |
| Explained Variance | Fraction of variance explained |

Cross-validation: `TimeSeriesSplit` (3 folds) for ML; fixed train/val/test split for DL.
